# selfpy

## 仮想環境の作成
   ・ python -m venv env または python3 -m venv env

## requestsのインストール
   ・ pip install requests

## インストールしたライブラリ「requirements.txt」の作成
   ・ pip freeze > requirements.txt

## インストールしたライブラリのインストール
   ・ pip install -r requirements.txt
